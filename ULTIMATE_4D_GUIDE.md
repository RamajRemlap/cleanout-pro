## üß† Ultrathink AI Integration

### What is Ultrathink?

Ultrathink is our advanced AI reasoning system that provides transparent, step-by-step analysis of room classifications.

### How It Works

1. **Click any room cube** in the visualization
2. **Ultrathink activates** automatically
3. **Watch the AI think** through 8 reasoning steps:
   - Analyzing room dimensions
   - Evaluating clutter density
   - Identifying item types
   - Assessing accessibility
   - Calculating workload
   - Cross-referencing data
   - Generating estimate
   - Completing analysis

4. **Review results**:
   - Size Classification
   - Workload Assessment
   - Confidence Score (0-100%)
   - Estimated Cost

### Confidence Visualization

The AI confidence is shown in multiple ways:
- **Color Intensity** - Higher confidence = brighter colors
- **Progress Bar** - Visual percentage indicator
- **Confidence Ring** - 3D ring around cube base
- **Particle Density** - More particles = higher confidence

## üêõ Troubleshooting

### Low FPS / Lag
**Solutions:**
- Reduce room count (filter jobs)
- Disable timeline (less rendering)
- Turn off auto-rotate
- Close background apps
- Update graphics drivers

### Cubes Not Appearing
**Solutions:**
- Check API connection
- Verify data loaded (see stats)
- Reload page (Ctrl+R)
- Check console (F12) for errors

### Fullscreen Not Working
**Solutions:**
- Try F11 instead of button
- Check Electron permissions
- Restart application
- Use browser version (temp)

### Ultrathink Not Showing
**Solutions:**
- Click room cube to trigger
- Enable AI toggle (üß† button)
- Check network connection
- Verify API health

### Colors Look Wrong
**Solutions:**
- Check monitor calibration
- Verify GPU color settings
- Try different browser/Electron
- Check accessibility settings
